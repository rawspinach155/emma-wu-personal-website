# Emma Wu's Personal Website

Emma Wu's interactive 3D personal portfolio, built with Next.js, React Three
Fiber, and Three.js.

## Prerequisites

- Node.js `>=22.13.0`

## Quick Start

```bash
pnpm install
pnpm dev
```

## Useful Commands

- `pnpm dev`: start local development
- `pnpm build`: create and verify a production Next.js build
- `pnpm start`: serve the production build locally

The site is configured for deployment on Vercel through the connected GitHub
repository.
